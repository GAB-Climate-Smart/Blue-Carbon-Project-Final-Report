# Prism Task Templates — Ecosystem Health Pilot

## Task 1: Structure harmonization
- Standardize heading hierarchy across all chapters.
- Remove duplicate paragraphs.
- Keep a clear result -> interpretation -> implication flow.

## Task 2: Readability pass
- Rewrite dense technical language into concise decision-oriented prose.
- Preserve all numbers and claims exactly unless source correction is provided.
- Define acronyms on first use.

## Task 3: Evidence-status tagging
- Ensure each section labels content as completed evidence, pending input, synthesis, recommendation, or assumption.
- Add bracketed placeholders where source-backed evidence is missing.

## Task 4: Recommendations QA
- Check that each recommendation is linked to at least one explicit finding.
- Flag and remove any unsupported recommendation language.

## Task 5: Final packaging
- Confirm figure/table callouts are sequential.
- Confirm unresolved issues are listed in limitations/pending section.
- Confirm no overclaim about EOSDA operational status.
