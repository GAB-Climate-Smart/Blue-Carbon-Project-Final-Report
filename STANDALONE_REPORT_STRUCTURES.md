## Framing Principle: The Four-Report System

These four reports are not independent products. They form an integrated evidence architecture:

```
Ecosystem Health Report   →  What condition are the wetlands in?
Carbon Inventory Report   →  What carbon stocks exist and at what rate are they at risk?
Drone / EOSDA Report      →  How are priority sites spatially defined and continuously tracked?
MRV Framework Report      →  How is all of the above monitored, reported, and verified nationally?
```

Each standalone is a primary technical reference AND a source extraction layer for the Blue Carbon Final Report. Every claim in the Final Report must trace to one of these four documents.

## REPORT 3: Carbon Inventory Assessment
### Status: PENDING — External Contributor | Structure defined here for handover

### Benchmark sources used to design this structure:
- IPCC 2013 Wetlands Supplement (Chapter 4: Coastal Wetlands)
- Verra VM0033 v2.1 — carbon pool accounting, stratification, and permanence
- CIFOR-ICRAF Mangrove Carbon Measurement Protocols (Howard et al.)
- High-Quality Blue Carbon Practitioners Guide (Ocean Risk Alliance, 2024)
- Ghana ToR Phase 4 requirements + MRV Framework carbon accounting section

### Why this structure:
Best-in-class blue carbon carbon inventory reports (Indonesia COREMAP-CTI, Madagascar WCS, Senegal Oceanium, Panama IBA) consistently follow a method chain logic: landscape context → stratification → pool selection → plot design → field measurement → calculation → stock summary → uncertainty → monitoring link. Verra/Gold Standard project descriptions require each step to be reproducible by an independent verifier. This structure ensures the Ghana report meets that bar.

## REPORT 4: Wetland Ecosystem Health Assessment
### Status: TO BUILD — from existing source report + monitoring plans

### Benchmark sources used to design this structure:
- Existing report: `ecosystem_report_keta_ada_final_edits_ef_17032026.md` (Dickson Yaw Agyeman PhD, January 2026)
- Monitoring plan: `monitoring_plan_for_keta_and_ada.md`
- Ramsar Convention Wise Use Concept and Ecological Character monitoring guidelines
- Millennium Ecosystem Assessment health indicators framework
- Ghana ToR Phase 2 requirements (ecosystem assessment of priority Ramsar sites)
- WACA ResIP 2 ecosystem health reporting expectations

### Why this structure:
The existing ecosystem health report is scientifically strong but formatted as an academic/technical report rather than a policy-and-management standalone deliverable. Best-in-class ecosystem health reports from comparable blue carbon assessments (Indonesia ICCAs, Senegal USAID mangrove, Madagascar Blue Ventures) expand the scientific findings with: explicit condition scoring or health categories, site-by-site comparative tables, management implication narratives, and direct linkage to the monitoring and MRV system. The standalone version needs to do all of that while preserving the full scientific rigour of the original.

### What the existing source report already provides (confirmed):
- Study area: Keta Lagoon Complex Ramsar Site + Songor/Ada
- 65 sampling stations (stratified random design, supervised GIS map)
- 270 water quality + 140 soil samples at Keta; 117 water quality + 28 soil samples at Ada
- Physico-chemical water parameters: pH, temperature, conductivity, salinity, TDS, DO, ORP, resistivity
- Soil chemical and biological indicators: nitrate, phosphate, potassium, pH, moisture, humus, temperature
- 9 reporting clusters at Keta: Anyanui–Dzita, Saviatula–Galotse, Aborlorvi–Norlopi, Afiadenyigba–Keta lagoon, Azizanya–Ayigbo
- 4 reporting clusters at Ada/Songor: Patukope–Pute, Futue–Luhue creeks, Wasakuse–Togbloku, Obane creek
- Key findings: pH 6.5–7.5 at Keta, temperature increase 3.2–6.1°C, elevated ORP at Ada, elevated potassium across sites
- Seasonal variability documented across all parameters
- Community change agent training completed

### What the standalone needs to add:
- Policy framing (Ramsar obligations, WACA mandate, NDC relevance)
- Ecosystem health condition scoring / classification by site
- Comparative site health table (Keta vs Ada/Songor)
- Linkage to carbon storage implications (healthy wetlands = higher SOC retention)
- Linkage to degradation pressure findings (cross-reference Environmental Pressure Assessment)
- Monitoring framework linkage (quarterly schedule, 57 + 23 stations, CCAs)
- Management recommendations structured for MLNR/MESTI/District use
